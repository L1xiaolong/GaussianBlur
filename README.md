### 结论  
#### 1. 核分离式高斯滤波能有效提高算法速度，配合NEON加速效果明显
#### 2. 高斯核尺寸小时，核分离是NEON速度与OpenCV相当；核尺寸大时，核分离NEON速度比OpenCV明显快
#### 3. 采用U8图像数据+定点化高斯核+NEON可进一步提升算法效率，其速度全面超越OpenCV，且效果与OpenCV一致
